# v1.0.0

## First public release

This release marks the first public version of the PromptPay QR Web App.

## Included in this release

- PromptPay QR web app with a minimal UI
- automatic QR generation when the required fields are filled
- Docker-based deployment using Nginx
- project structure organized into `public/` and `deploy/`
- English `README.md`
- `LICENSE` file
- `SECURITY.md` policy
- live deployment example through Docker and Cloudflare Tunnel

## Deployment summary

- Static front-end served by Nginx
- Docker image for repeatable deployment
- Cloudflare Tunnel support for exposing the app publicly

## Notes

- This release is intended as the first public baseline
- Future releases can build on top of this version with UI improvements, additional documentation, or new features
